================================================================================
  CC Explore v2.0 — Installation Guide
  CC的探讨 v2.0 — 安装说明
================================================================================


  ⏱  3 steps · < 30 seconds · Windows / macOS / Linux
  ⏱  3 步 · 不到 30 秒 · 全平台支持


================================================================================
  Requirements / 环境要求
================================================================================

  • Claude Code CLI (any version / 任意版本)
  • OS: Windows / macOS / Linux
  • Disk: < 50KB (just two markdown files / 仅两个 markdown 文件)


================================================================================
  Step 1 — Get the files / 获取文件
================================================================================

  git clone https://github.com/yyyxxx-art/cc-explore.git
  cd cc-explore

  Or download ZIP and extract anywhere.
  或直接下载 ZIP 解压到任意位置。


================================================================================
  Step 2 — Install Skills / 安装 Skills
================================================================================

  # Create skills directory if not exists
  # 创建 skills 目录（如果还没有）
  mkdir -p ~/.claude/skills

  # Copy both skill files
  # 复制两个 skill 文件
  cp skills/cc-save.md ~/.claude/skills/
  cp skills/cc-load.md ~/.claude/skills/


================================================================================
  Step 3 — Verify / 验证安装
================================================================================

  Open Claude Code and type / 打开 Claude Code，输入：

    /cc-load

  If you see the workflow list or "no archives yet", it's working. ✅
  如果显示出工作流清单或提示尚无存档，即为安装成功。✅


================================================================================
  Uninstall / 卸载
================================================================================

  rm ~/.claude/skills/cc-save.md
  rm ~/.claude/skills/cc-load.md

  The Desktop "CC的探讨" folder is NOT auto-deleted (your archives are safe).
  Remove it manually when you're done.
  桌面 CC的探讨 文件夹不会被自动删除（对话存档保留），手动删除即可。


================================================================================
  Optional: Archive folder paths / 可选：存档路径
================================================================================

  Default archive location / 默认存档位置:

    Windows:  C:\Users\{username}\Desktop\CC的探讨\
    macOS:    ~/Desktop/CC的探讨/
    Linux:    ~/Desktop/CC的探讨/

  To change, edit path references in skills/cc-save.md and skills/cc-load.md.
  如需修改，编辑 skill 文件中的路径引用。


================================================================================
  Optional: Pre-create the Desktop folder / 可选：提前创建桌面文件夹
================================================================================

  mkdir -p ~/Desktop/CC的探讨

  (First /cc-save will auto-create it anyway. / 首次使用时也会自动创建。)
